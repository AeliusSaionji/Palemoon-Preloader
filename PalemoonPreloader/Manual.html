Introduction

This is a little program for people who use Palemoon on older machines. It is designed to pre-load Palemoon in the background so it will load faster when requested.

Requirements

  • Palemoon 1.0 or better
  • 4MB plus Enough memory to run Palemoon
  • Window 95 or NT 4.0 or better

Installation

Once you have downloaded the installation program, simply run this program and it will guide you through installing the Palemoon Preloader. If you are running an older version, it is recommended that you unload it or you will have you restart your computer when the installation finishes.

The first time the pre-loader runs, it may ask you for the directory that Palemoon is installed in if it was unable to automatically find Palemoon. You may change this later if you wish.

Using the Preloader

To use the Palemoon Preloader, simply place a shortcut in the Windows StartUp folder, if one is not present. To begin using if right away, simply use the shortcut in the Windows Start Menu under the Palemoon Preloader program group.

When the Preloader is loaded, a Palemoon icon will appear in the system tray. By right-clicking on the icon, you can bring up a menu of actions to control Palemoon and the Preloader. By double-clicking on the icon, you can open a new Palemoon window with your currently set home page(s).

The Palemoon Preloader will immediately load a portion of Palemoon into memory to cut the startup time of Palemoon in half. As you begin to use Palemoon the remaining portions will be loaded into memory. When all of the Palemoon Windows are closed, Palemoon will remain in memory and the next startup will be instantaneous.

If you use the ‘File->Exit’ menu item in any Palemoon browser windows, Palemoon will be unloaded and all the memory it was using freed. The Preloader will then reload Palemoon, making it available for later use.

Installing and Removing Themes and Extensions

Since the Palemoon Preloader will keep Palemoon loaded, even when no Palemoon windows are visible, installing or removing themes and extensions may not take affect as expected. Since Palemoon must be restarted to remove or install a theme or extension, you must terminate Palemoon using either of the following methods:

  • Use the ‘File->Exit’ menu item in any Palemoon windows.
  • Use the “Reload Palemoon” menu item in the Preloader's tray icon menu.

Should you plan to work or/develop a theme or extension, the Preloader can easily get in the way be requiring the extra effort to terminate Palemoon. To make sure that Palemoon always unloads when you close all Palemoon windows, you can deactivate the Preloader by using the “Unload Palemoon” menu item from tray icon menu. You can make the Preloader resume its normal function by using the “Reload Palemoon” menu item in the tray icon menu.

Using the Preloader Menu

Right clicking on the Palemoon icon in the tray will present the following menu items:

  • Open My Homepage - Opens your current home page in a new browser window
  • Internet Options - Opens the preferences window to set Palemoon's options
  • Preloader Options - Opens the dialog where you set the Palemoon path and other options the pre-loader needs
  • Reload Palemoon - Unloads (if pre-loaded) and reloads Palemoon to clean Palemoon's memory cache, enable newly installed extensions, and use a newly applied theme. This also reactivates the Preloader if it was deactivated.
  • Unload Palemoon - Unloads Palemoon and deactivates the Preloader to free memory for other applications, use “Reload Palemoon” to reactivate the Preloader and load Palemoon into memory. This options is disabled when the Preloader is inactive.
  • About Palemoon Preloader - Displays the current version of the Palemoon Preloader installed
  • Unload Palemoon Preloader - Unloads the Palemoon Preloader and any instances of Palemoon loaded by the Preloader
